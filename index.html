General Web, CSS and Bootstrap Resources

  • COLOURlovers — a great resource for color palettes.
  • Computer Hope's HTML color codes & names — this is an awesome resource for shades of a color — you can type a hexidecimal color code in the box at the top and it will show you a list of lighter and darker shades and other useful things.
  • w3schools CSS Reference — one of the more user-friendly CSS references out there.
  • Bootstrap 3.3.7 CSS reference — because you need this. Really.
  • Bootstrap 4 reference — because this is what CodePen loads by default and if you're not going back to 3.3.7, you need this.
  • Lorem Ipsum — because.
  • W3C Markup Validation Service — you can paste your HTML here and it will tell you if it's valid. Useful.
  • pixabay — free, high quality stock images. That's right. Free to use. Donations are encouraged.

Tribute Page Resources

Portfolio Resources
