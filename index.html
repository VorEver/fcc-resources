General Web, CSS and Bootstrap Resources

  • COLOURlovers — a great resource for color palettes.
  • Computer Hope's HTML color codes & names — this is an awesome resource for shades of a color — you can type a hexidecimal color code in the box at the top and it will show you a list of lighter and darker shades and other useful things.
  • Material Palette — lets you pick two colors and generates a palette for you.
  • Colour Hunt Palettes — another good source of ready-made palettes.
  • Materialette — a CodePen color palette gadget.
  • HTML Color — groups colors (greens, blues, etc.) and also helps you find information on colors if you search on the hexcode.
  • w3schools HTML5 reference — easy to understand.
  • HTML5 Reference — the real deal.
  • HTML5 Basics videos — 7 videos to help you learn HTML5 basics.
  • w3schools CSS Reference — one of the more user-friendly CSS references out there.
  • CSS Reference — the real deal.
  • CSS Basics videos — 8 videos to help you learn CSS basics.
  • Bootstrap 3.3.7 CSS reference — because you need this. Really.
  • Bootstrap 4 reference — because this is what CodePen loads by default and if you're not going back to 3.3.7, you need this.
  • DevTips: The Bootstrap Grid videos — a series of awesome videos on how to use Bootstrap.
  • Lorem Ipsum — because.
  • W3C Markup Validation Service — you can paste your HTML here and it will tell you if it's valid. Useful.
  • pixabay — free, high quality stock images. That's right. Free to use. Donations are encouraged.
  • Unsplash — free high-resolution photos.
  • THE STOCKS — free images, colors, icons, etc.
  • ISO REPUBLIC — free stock photos.
  • Subtle Patterns — subtle background textures.
  • CSS Tricks — good site for little tidbits of CSS know-how.
  • Table of HTML Codes — HTML codes for common symbols and stuff.

Tribute Page Resources

  • Wikipedia — usually you can find something about someone here.
  • postimage — if they aren't on the Internet (like your grandmother or your cat), you can put their picture here.
  • Google Fonts — Lobster isn't the only Google font — find others here.
  • Font Squirrel — ... and Google isn't the only source of fonts.
  • How to Directly Link or Embed DropBox Images — so you don't have to host it somewhere.

Portfolio Resources

  • GitHub Pages — you can put your portfolio as your GitHub page. Choose [User or organization site] from the options above step 1. Follow the directions. You must have the git software installed.
  • GitHub for Noobs! — YouTube videos to help you learn GitHub.
  • Font Awesome Icon intro on w3schools — learn how to use font awesome (or other) icons.
  • Font Awesome Icons — the actual icons.
  • ionicons — an icon set.
  • flaticon — almost half a million icons. That's a lot.
  • iconmonstr — searchable collections of icons.
  • One Page Love — the ultimate showcase of one-page websites. If you don't want your portfolio looking like everyone else's.

Quote Machine Resources

  • CHUCKNORRIS.IO — random Chuck Norris quotes API.
  • Quotes on Design — design quotes API.
  • Beau Teaches Javascript — this is an amazing set of javascript tutorials. If you're stuck on something, you'll find a lesson here that can help.
  • How to Share Webpages with Twitter — this helped me.
  • publish.twitter.com — Twitter's helpful page.
  • Learn jQuery in 15 Minutes — 4 videos to help you learn jQuery.
  • Javascript Reference — this will help you if you know what you're looking for. Bookmark it.
  • Javascript Operator Precedence — which comes first? From parentheses to commas, it's in there.
